# Sales Analysis Dashboard

## Project Overview
This project analyzes sales data using MySQL, Excel, and Power BI. The objective is to identify sales trends, profit performance, top-performing states, and customer payment preferences through interactive visualizations.

## Tools Used
- MySQL
- Excel
- Power BI
- GitHub

## Key Performance Indicators (KPIs)
- Total Sales: 6M
- Total Profit: 2M
- Total Quantity Sold: 13K

## Dashboard Features
- Sales by State
- Sales by Category
- Sales by Payment Mode
- Yearly Sales Trend
- Interactive Filters

## Key Insights
- New York generated the highest sales among all states.
- Electronics and Office Supplies were major contributors to revenue.
- Sales reached their peak in 2022.
- Multiple payment methods were used with balanced distribution across customers.
